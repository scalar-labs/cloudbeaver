# Command

Click with a mouse or keyboard to trigger an action. This abstract component is based on the WAI-ARIA Command Role

## Props

This component forwards props to Ariakit's `Command` component and accepts common React HTML attributes.

- `disabled` — Boolean to disable interaction. Disabled command is not clickable or selectable via keyboard

Underlying component docs:
[https://ariakit.org/reference/command](https://ariakit.org/reference/command)
